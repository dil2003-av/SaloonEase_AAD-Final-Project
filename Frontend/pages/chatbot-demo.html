SaloonEase Chatbot Demo

Chatbot Setup Complete!

Your SaloonEase chatbot has been successfully integrated into your salon website. Here's what's been set up:

  • Backend Integration: Connected to Ollama with gemma3:1b model
  • Frontend Widget: Beautiful floating chatbot widget
  • Pages Updated: Customer Dashboard, Admin Dashboard, Services, About, Gallery, and Appointments
  • Salon-Specific Context: AI assistant trained for salon services and booking assistance
To start using the chatbot:
  1. Make sure Ollama is running on your system: ollama serve
  2. Verify the gemma3:1b model is installed: ollama pull gemma3:1b
  3. Start your Spring Boot backend server
  4. Open any of the updated pages to see the chatbot in the bottom-right corner
Configuration Details
ollama.api.url=http://localhost:11434/api/generate
ollama.model=gemma3:1b
Backend API: http://localhost:8080/api/chat
